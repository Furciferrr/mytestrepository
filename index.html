Команды GIT'а

  • git clone https... - клонировать репозиторий на комп
  • git add index.html - добавить (сказать гиту, чтоб трекал) новый созданный файл
  • git status - проверить статус (показать изменения)
  • git commit -am 'название изменений' - зафиксировать изменения и дать им коментарий
  • git push - отправить все закомиченные изменения на сервер
  • git pull - забрать изменения с сервера
